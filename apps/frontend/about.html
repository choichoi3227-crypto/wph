About

왜 CloudPress Bridge인가

CloudPress Bridge는 한 가지 전제에서 시작했습니다. 자본금 제로, 외부 의존 제로로도 프로덕션 수준의 워드프레스 호스팅을 운영할 수 있다.

기술 스택에 대하여

Cloudflare Workers, KV, Durable Objects(SQLite), PHP-WASM만으로 구성됩니다. R2도, AWS S3도, 외부 MySQL도, GitHub 저장소도 없습니다. 파일은 자체 제작 S3 호환 스토리지(KV 청크 엔진)에, DB는 Durable Object 내장 SQLite에, 그리고 워드프레스 런타임은 Workers 위의 PHP-WASM에 올라갑니다.

아키텍처 선택에 대하여

Durable Objects는 Durable Objects대로, KV는 KV대로 각자 잘하는 일에 집중합니다. 메타데이터 인덱스는 DO 내장 SQLite가, 실제 바이트 데이터는 KV 청크 엔진이, 관계형 사용자/결제 데이터는 D1이 담당합니다. Workers 10개를 호스팅당 할당해 LoadBalancerDO가 헬스체크와 라운드로빈을 처리합니다.

가격에 대하여

라이트 $15, 스탠다드 $30, 스마트 $49. 호스팅 단위 월정액입니다. DNS는 Zone당 $0.10, 쿼리당 $0.09를 추가로 부과합니다. 모든 플랜은 PayPal 단건 결제로 처리됩니다.
