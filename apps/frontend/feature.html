스토리지

100% 자체 S3 호환 스토리지

R2도, AWS S3도 아닙니다. Workers + KV + Durable Object만으로 구현한 오브젝트 스토리지. SigV4와 Bearer 토큰 인증을 모두 지원해 기존 S3 클라이언트를 그대로 사용할 수 있습니다.

  • ✓  4MB 청크 분산 저장 (KV 25MB 한도 대응)
  • ✓  멀티파트 업로드 지원
  • ✓  HTTP Range 요청 지원
  • ✓  AWS SigV4 + Bearer 이중 인증
  • ✓  ListObjectsV2 호환
AWS S3 호환 PUT
# 기존 aws-sdk 그대로 사용 가능
const s3 = new S3Client({
  endpoint: "https://storage.cloud-press.co.kr",
  credentials: { accessKeyId, secretAccessKey }
});

await s3.send(new PutObjectCommand({
  Bucket: "my-site",
  Key: "wp-content/photo.jpg",
  Body: fileBuffer
}));
